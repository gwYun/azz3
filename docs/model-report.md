# 축구 이적료 예측 모델 — 기술 보고서

| 항목 | 내용 |
|---|---|
| 과제 | 직전 시즌 FBref 스탯 → Big-5 인바운드 **이적료(EUR)** 회귀 예측 |
| 알고리즘 | XGBoost (log1p-deflated 타깃), 베이스라인 Ridge |
| 현재 성능 (홀드아웃 n=292) | **MAE €5.35M, Spearman ρ 0.846** |
| 학습 모집단 | Big-5 인바운드·공개료·대여제외 이적 2014–2022, **조인 2,563행** |

> 산출물: `data/models/xgb_transfer_fee.pkl`, `data/models/selected_features.json`

---

## 1. 모델·학습 데이터 및 데이터 적법성

**데이터 파이프라인** — 단일 출처 `JaseZiv/worldfootballR_data`(GitHub) RDS 덤프 8개(~12MB)를 순수 파이썬 `rdata`로 읽어, **라이브 스크래핑 없이** 정적 다운로드합니다(`src/data.py`). 세 레이어를 결합:

- **FBref 6개 테이블**(standard/shooting/passing/possession/defense/misc), 2010–2023, Big-5
- **Transfermarkt 이적**(공개료 약 24,000건, EUR, 2010–2022) + **시장가치 스냅샷**
- 결합 키: TM↔FBref 공통 ID가 없어 **정규화 이름 + 직전 시즌 + 나이(±2년)** 퍼지 매칭. Big-5 인바운드 4,775건 → 직전 시즌 스탯 보유분 **2,563행**(46.3% 탈락).

**데이터 적법성** (법률 자문 아님, 실무적 평가)

- 출처 데이터는 FBref(Sports Reference/StatsBomb)·Transfermarkt에서 파생된 **커뮤니티 재배포본**입니다. 두 원천 모두 ToS상 대량 스크래핑·상업적 재배포를 제한합니다.
- 본 프로젝트는 **라이브 스크래핑이 아닌 정적 공개 덤프**를 사용하고, 데이터가 **사실(경기 기록·이적료)** 이라 저작물성이 낮습니다 → **비상업적·연구/포트폴리오 MVP로는 통상적이고 위험이 낮은** 범주.
- 단, **상업 배포 시에는** 정식 라이선스(Opta/StatsBomb/Transfermarkt 상업 계약) 검토가 필요합니다. EU 데이터베이스권(sui generis)·GDPR(선수 데이터는 공개·직업적 정보이나 여전히 개인정보)도 상업화 단계에서 점검 대상입니다.

## 2. 모델 특성·피처 선정·영향력·사용 정당성

**피처 선정 기준** (`src/features.py`) — **LASSO + RFE + 상호정보량(MI) 3종 앙상블 투표**로 80여 후보 중 30개 선정 + 강한 예측자가 탈락하지 않도록 **도메인 필수 포함**. 데이터 누수 방지를 위해 **시간 분할 + 선수 disjoint**(train에 나온 선수는 test에서 제거, assert 검증)를 강제합니다(`src/model.py`).

**피처 영향력 (현재 모델 gain% 실측, 상위)**

| feature | gain% | 해석 |
|---|---|---|
| prior_market_value_eur | 27.8% | TM 시장가치가 단일 최강 신호(앵커) |
| buyer_premium_x_value | 7.4% | 바이어 구단 프리미엄 × 가치 상호작용 |
| age_years | 7.1% | 나이 — 가장 강한 **하향** 레버 |
| league_premier_league | 7.0% | 출신 리그 효과 |
| tenure_at_selling_club_years | 4.3% | 잔류 기간 |

(30개 전부를 트리가 사용. 전역 중요도 외 **선수별 SHAP**도 내장 — `src/shap_utils.py` `TreeExplainer` + 웹 API `top_3_perturbations`.)

**모델 사용 정당성**

- 이적료는 스탯에 **강한 비선형·상호작용**(나이 곡선, 슈퍼스타 프리미엄, 한계 체감, 두꺼운 꼬리)을 가지므로 선형 모델로 부족합니다. 동일 피처·동일 분할에서 **XGBoost €5.35M vs Ridge €12.37M(MAE −57%), ρ 0.733→0.846** — 트리 모델 채택이 실증적으로 정당화됩니다.
- 테이블형 회귀에서 GBM 계열은 여전히 사실상 SOTA(Grinsztajn 2022 등)이며, 보수적 하이퍼파라미터(depth 4·400트리·lr 0.05·subsample 0.8)는 ~2,500행 규모에 적합합니다.

## 3. 주요 예측 결과

**집계 성능 (현재 모델, 홀드아웃 n=292):** MAE €5.35M, Spearman 0.846, drift 분리 추적.

**예측 거동** (홀드아웃 사례로 본 패턴):

- **중상위권은 정확** — Wesley Fofana 실제 €80.4M / 예측 €86.0M(오차 7%), Erling Haaland €60.0M / €66.2M(10%).
- **메가딜은 체계적 과소예측** — Jack Grealish €117.5M / €74.2M(−37%), Casemiro €70.7M / €34.4M(−51%). 모델이 못 보는 **"슈퍼스타·협상 프리미엄"** 성분이 잔차의 주원인.
- **반사실 일관성** — 거의 모든 선수에서 `-age_years`(나이 감소)와 `+SoT_per_90`(유효슈팅)이 이적료 상승 1순위로 나와, 도메인 직관과 부합.

## 4. 확장성 — 데이터 증가 시 학습 한계 전망

**실증 근거 (이미 관측됨):** 학습 데이터를 **551행 → 2,563행**으로 늘렸을 때 **MAE €14.4M → €5.35M(−63%), ρ 0.740 → 0.846**. 즉 이 과제는 현재 **데이터 양에 강하게 반응**하는 구간입니다.

**5개국 전 리그(하부 포함) 확장 전망**

- **데이터 양**: 조인 행 ~2,563 → 낙관적 **8,000–15,000행**. 단 하부는 무공개료 비중이 커 선형적으로 늘지 않음.
- **필수 추가 데이터**: ① 하부 리그 **고급 스탯 커버리지**(현 소스로는 결측 많음), ② **리그 강도 지표**(없으면 티어 간 스탯 비교 불가 — 확장의 전제조건), ③ 무공개료를 다루는 **검열(censored) 회귀**.

**예상 학습 한계 (수렴 지점)**

- 데이터·피처를 충분히 보강하면 MAE는 더 내려가나 **체감 수렴**합니다. 환원 불가능한 오차 하한은 (a) 슈퍼스타/협상·여론 프리미엄, (b) 무공개료 검열, (c) 타깃이 시장가치와 부분적으로 순환 — 이 세 가지가 R²의 천장을 만듭니다.
- 현실적 전망: 부상 이력·계약/방출조항·구단 재무·리그 강도까지 더하면 **상위권 MAE를 한 자릿수 €M 초반대**로 낮추고 순위 상관 **ρ ~0.88–0.92** 수준에 접근 가능. 다만 메가딜의 프리미엄 성분은 스탯만으로 완전히 설명되지 않아, **"순위 매김"은 매우 정확해도 "절대 금액"에는 구조적 잔차**가 남습니다.
